# Splash: AI 原生 UI 脚本语言

## 执行摘要

**Splash** 是一款为 AI 时代打造的革命性脚本语言，能够在 Makepad 框架中实现自然语言驱动的 UI 生成。它代表了人机交互的下一次进化：用户描述需求，系统实时构建界面。

---

## 愿景

### 从代码到对话

传统 UI 开发需要：
- 学习复杂的编程语言
- 理解特定框架的 API
- 编写数百行样板代码

**Splash 改变了一切。**

使用 Splash，创建 UI 就像说话一样简单：
```
"创建一个带有邮箱和密码字段的登录表单"
"添加一个包含三个指标卡片的仪表板"
"构建一个带有消息历史的聊天界面"
```

AI 理解用户意图，生成最优的组件结构，并即时渲染。

## 技术差异化优势

### 1. 原生性能，动态灵活

与基于 Web 的解决方案（React、Flutter Web）不同，Splash 运行在 **Makepad** 之上，这是一个 GPU 加速的原生框架：

| 指标 | React | Flutter | Splash/Makepad |
|-----|-------|---------|----------------|
| 包体积 | 150KB+ | 2MB+ | 800KB |
| 首次绘制 | 500ms+ | 300ms+ | <50ms |
| 60fps 保证 | 否 | 部分 | 是 |
| 内存占用 | 高 | 中 | 低 |

### 2. Rust 基础 = 安全 + 速度

- **内存安全**：无缓冲区溢出，无空指针异常
- **无畏并发**：设计上安全的多线程
- **零 GC 暂停**：一致的帧时间，对实时 UI 至关重要

### 3. 原生跨平台

单一代码库部署到：
- macOS、Windows、Linux（原生桌面）
- iOS、Android（移动端）
- WebAssembly（浏览器）
- 嵌入式 Linux（IoT/汽车）
